# 快速上手指南

> 这是你的**短视频内容运营 Agent**，专为抖音/视频号个人成长/心理学方向的创作者打造。
>
> **你只需要做两件事**：
> 1. 填写 `soul/my-soul.md`（告诉 Agent 你是谁、你想做什么内容）
> 2. 和 Agent 对话，让它帮你写脚本、找选题、分析热点
>
> 这份指南面向完全没有编程基础的用户，每一步都写清楚具体操作。

---

## 第一步：安装 Claude Code

> 如果你已经在用 Claude Code（就是现在这个界面），跳过这一步。

1. 打开浏览器，访问 `https://claude.ai`
2. 注册或登录你的账号
3. 下载 Claude Code 桌面应用并安装（Windows 或 Mac）
4. 安装完成后打开，用账号登录

---

## 第二步：安装两个插件（可选，增强功能）

插件安装后，Agent 能记住你们以往的所有对话，定时帮你推送热点早报和内容计划。

在 Claude Code 底部输入框里，逐条复制粘贴以下内容并按回车：

```
/plugin marketplace add thedotmack/claude-mem
```
```
/plugin install claude-mem
```
```
/plugin marketplace add jshchnz/claude-code-scheduler
```
```
/plugin install scheduler@claude-code-scheduler
```

安装完成后，关闭 Claude Code 再重新打开，让插件生效。

---

## 第三步：打开项目

1. 打开 Claude Code
2. 点击左上角的文件夹图标
3. 选择这个项目文件夹（就是包含 `QUICKSTART.md` 的文件夹）
4. 打开后左侧会显示所有文件目录

---

## 第四步：填写 my-soul.md（最重要）

这个文件告诉 Agent 你是谁。填得越详细，Agent 写出来的脚本越符合你的风格。

### 怎么打开：
在左侧文件目录里，点 `soul` 文件夹 → 点 `my-soul.md`

### 怎么填：
文件里每个模块都有灰色的注释说明（`<!-- ... -->` 包起来的内容），告诉你填什么。灰色注释不会被 Agent 看到，删不删都行。

**至少先填这几个（其他的边用边填）**：

**我是谁（创作者身份）**：
```
我是一个经历过长期焦虑、最近开始学习心理学的普通人，
想用自己走出来的经历帮助同样困扰的人。
```

**目标受众**：
```
20-30岁有焦虑、容易内耗、想改变但不知从哪里开始的年轻人
```

**内容定位**：
```
情绪管理 + 打破内耗的实用方法，不说大道理，只讲可以立刻用的方法
```

**内容风格**：
```
温柔但直接，像一个懂点心理学的朋友，用自己的故事带出观点
```

填完后按 `Ctrl+S`（Windows）或 `Cmd+S`（Mac）保存。

---

## 第五步：开始和 Agent 对话

在 Claude Code 底部的输入框里，直接告诉 Agent 你想做什么。

### 常用对话方式：

**让 Agent 写脚本**：
```
帮我写一条关于"内耗"的视频脚本，60秒以内，
目标受众是经常自我批评、做什么都觉得不够好的人
```

**让 Agent 给你几个钩子变体**：
```
我想做一条关于"讨好型人格"的视频，帮我写5种不同风格的开场钩子，
我来选一个最顺手的
```

**让 Agent 帮你策划一个内容系列**：
```
帮我策划一个围绕"情绪管理"的系列，大概8-10条，
给我每条的选题和钩子方向
```

**让 Agent 分析最近的热点**：
```
最近抖音上个人成长/心理学方向有什么热门话题？
帮我找3个可以切入的角度
```

**让 Agent 分析竞品账号**：
```
帮我分析一下抖音上做心理学内容的账号，
他们常用什么钩子，我怎么做得不一样
```

## 第六步：往 data/ 里扔数据（让 Agent 越来越懂你）

数据扔得越多，Agent 生成的脚本越符合你的风格。

### 可以扔什么：

| 数据 | 放哪里 | 格式 | 效果 |
|------|--------|------|------|
| 你写过的脚本/文案 | `data/raw/scripts/` | 任意文本文件 | Agent 学习你的语言风格 |
| 你喜欢的账号的内容 | `data/raw/competitors/` | 文本文件 | Agent 分析风格，帮你差异化 |
| 你的读书笔记/心得 | `data/raw/notes/` | 文本文件 | 成为脚本内容的素材 |
| 你觉得写得好的文章 | `data/raw/references/` | 文本文件 | 丰富内容深度 |

### 操作方法：
1. 打开文件管理器，找到对应文件夹
2. 把你的文件直接拖进去
3. 回到 Claude Code，告诉 Agent：

```
我往 data/raw/scripts/ 里放了几个我自己之前写的文案，
帮我分析一下我的写作风格，然后按这个风格帮我写下一条脚本
```

---

## 第七步：填写视频数据反馈（让 Agent 越来越准）

每次发布视频后，把数据记录进来，Agent 会用这些数据做复盘，告诉你什么内容有效。

1. 在左侧文件目录里，找到 `data/feedback/ratings.json`
2. 打开文件，按文件里的例子格式，把你的视频数据填进去
3. 或者直接告诉 Agent：

```
我刚发的那条关于内耗的视频，播放量1200，完播率45%，
点赞89，评论区很多人说"说的就是我"。
帮我分析一下效果，下次怎么改进？
```

---

## 第八步：查看 Agent 的产出

Agent 所有产出都保存在 `output/` 文件夹里：

| 文件夹 | 内容 |
|--------|------|
| `output/scripts/` | 生成的视频脚本 |
| `output/hooks/` | 钩子变体库 |
| `output/series/` | 内容系列策划 |
| `output/calendar/` | 热点早报、周计划 |

在左侧文件目录点开 `output/` 文件夹，点击任意文件即可查看。

---

## 第九步：查看定时任务

已为你注册了 3 个自动运行的云端任务：

| 任务 | 运行时间 | 内容 |
|------|---------|------|
| 热点早报 | 每天 09:30 | 推送今日个人成长/心理学相关热点和脚本灵感 |
| 周内容规划 | 每周一 09:00 | 生成本周 3 条视频的选题+钩子方向 |
| 竞品观察 | 每周四 10:00 | 分析本周赛道爆款，提炼对你的启发 |

在 claude.ai/code/routines 可以查看和管理这些任务。

## 常见问题

**Q：我完全没做过视频，Agent 能帮我做什么？**
A：最重要的是脚本。让 Agent 帮你写脚本，你只需要照着脚本说话录制就行。从第一条开始，不要等到"准备好了"。

**Q：Agent 写的脚本听起来不像我说话的方式怎么办？**
A：两个方法：1）在 `my-soul.md` 里把你的风格描述得更具体；2）把你自己写的文案放进 `data/raw/scripts/`，让 Agent 学你的语言习惯。

**Q：怎么让脚本更适合抖音/视频号的风格？**
A：告诉 Agent 具体要求，比如：
```
用更口语化的方式改写这个脚本，
句子要短，像朋友聊天，不要书面语，
特别是开头3秒要更有冲击力
```

**Q：定时任务的结果在哪里看？**
A：定时任务是云端运行的，结果会出现在 claude.ai 的对话里，不在你本地电脑上。你可以在 claude.ai/code/routines 查看每次运行的结果。

**Q：我现在最应该做什么？**
A：按这个顺序：
1. 填写 `soul/my-soul.md`（10分钟）
2. 让 Agent 帮你写第一条脚本
3. 把脚本拍出来发出去
4. 把数据告诉 Agent，让它帮你复盘
5. 写第二条

**不要在"准备"上花太多时间，第一条视频比完美的策划更重要。**

## 脚本结构

### 【开场钩子】（0:00 - 0:30）

> 目标：前 3 秒抓住注意力，30 秒内让观众决定是否继续看

```
[震撼数据 / 反直觉观点 / 悬念问题]

"今天我要告诉你 [核心结论]。
听完这个视频，你会对 [主题] 有完全不同的认识。"
```

**B-roll 建议**：{{BROLL_HOOK}}

### 【背景铺垫】（0:30 - 2:00）

> 目标：建立共同认知，定义这个问题为什么重要

```

[数据支撑为什么这件事重要]
[与观众切身利益的连接]
```

### 【风险与反驳】（{{RISK_START}} - {{RISK_END}}）

> 展示客观性，增强可信度

```
"当然，这个逻辑也有漏洞。[反方观点]"

"我仍然认为 [结论] 的原因是 [理由]。"

"如果 [风险情景] 发生，我会 [应对方式]。"
```

---

### 【结论与行动建议】（{{CONCLUSION_START}} - {{CONCLUSION_END}}）

```
"总结一下：[一句话核心结论]"

"基于这个分析，我的行动框架是：
• [情景 A] 时，[行动 X]
• [情景 B] 时，[行动 Y]"

"⚠️ 以上仅为个人分析和投资记录，不构成投资建议。
每个人的风险承受能力不同，请独立判断。"
```

---

### 【结尾互动】（最后 30 秒）

```
"你对 [主题] 怎么看？评论区告诉我。"

"如果这期视频对你有帮助，点赞和订阅是对我最大的支持。"

"下期我会讲 [预告主题]，感兴趣的记得开启通知。"
```

# 推文模板

> Agent 使用说明：参考 data/raw/tweets/ 中的历史推文提炼用户写作风格，结合以下结构生成草稿，保存到 output/content/。
> 推文字符限制：Twitter/X 单条 280 字符（英文）/ 约 140 汉字。

---

## 模板 A — 数据驱动型（适合行情播报/信号提醒）

```
[核心数据/结论，一句话]

[支撑数据 1]
[支撑数据 2]
[支撑数据 3]

[1-2 句观点或行动提示]

#标签1 #标签2
```

**示例**：
```
BTC RSI 跌到 28，上次这个水平是 2022 年 12 月底。

当前抄底信号：
• RSI(14): 28 ✅
• 加密 FGI: 12（极度恐惧）✅
• MVRV Z-Score: -0.3 ✅

3/5 指标触发，关注区间开始。不是底部信号，是开始关注的信号。

#BTC #加密货币
```

---

## 模板 B — 观点驱动型（适合宏观判断/投资逻辑）

```
[一个反直觉或有洞见的观点，抓眼球]

[论据 1]
[论据 2]
[论据 3]

[结论/行动建议]
```

## 模板 C — 事件评论型（适合财报/政策/市场事件）

```
[事件名称] 核心点：[一句话总结]

利好：[...]
利空：[...]
市场预期 vs 实际：[...]

我的判断：[一句话，有立场]

#标的代码 #相关标签
```

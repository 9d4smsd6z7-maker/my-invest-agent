# 短视频脚本模板（60秒内）

> Agent 使用说明：参考 data/raw/scripts/ 中的历史脚本提炼用户语言风格，结合以下结构生成完整脚本。
> 60秒内：约 200-250 字。说话速度约 250字/分钟，抖音/视频号完播率最佳区间。

---

## 基本信息

- **选题**：{{TOPIC}}
- **目标受众**：{{TARGET}}（具体一个人的状态）
- **核心观点**：{{CORE_POINT}}（一句话）
- **情绪基调**：{{TONE}}（温柔/犀利/共鸣/反常识）
- **预计时长**：{{DURATION}}

---

## 脚本正文

### 【钩子】（0-3秒，约 15-20 字）

> 目标：让人停下来。一句话，引发好奇、共鸣或反常识感。

```
{{HOOK}}
```

**备选钩子**（Agent 额外给 2 个变体）：
- 变体 A：{{HOOK_A}}
- 变体 B：{{HOOK_B}}

---

### 【痛点/共鸣】（3-15秒，约 40-50 字）

> 目标：让用户说"说的就是我"，情绪连接建立，用户决定看下去。
> 描述一个具体场景，不要抽象概念。

```
{{PAIN_POINT}}
```

---

### 【价值交付】（15-50秒，约 120-150 字）

> 目标：核心内容。干货/故事/方法，让用户觉得"这个我没想到/这个对我有用"。
> 分 2-3 个层次，每层一句铺垫 + 一句核心。

```
{{MAIN_CONTENT}}
```

---

### 【金句收尾】（最后 5-8 秒，约 15-20 字）

> 目标：一句可以截图传播的话。可记忆，有力量，与选题高度相关。

```
{{CLOSING_LINE}}
```

---

### 【CTA】（自然嵌入，不生硬）

> 选一个，不要全用：

- 评论互动：`你有过这种感觉吗？评论区告诉我`
- 关注引导：`更多这样的内容，关注我`
- 收藏提醒：`这条建议先收藏，以后用得上`
- 转发号召：`如果你身边有人需要听到这些，转给他/她`
